Assignment 3: Data Visualization (incarceration)

Introduction

Racism is a big problem in the United States. In this project, I attempted to analyze incarceration trends to compare the incarceration rates between black and white populations.

Summary

For my variables, I chose to analyze highest_pop_year, highest_pop_state_1993, highest_pop_state, highest_black_pop_state, and highest_black_pop_county. These variables helped me decide the most adequate region I should use to compare the incarceration rates of two populations.

highest_pop_year <- data %>%
  filter(total_jail_pop == max(total_jail_pop, na.rm = TRUE)) %>%
  pull(year)

highest_pop_state_1993 <- data %>%
  filter(year == "1993") %>%
  filter(total_jail_pop == max(total_jail_pop, na.rm = TRUE)) %>%
  pull(state)

highest_pop_state <- data %>%
  filter(year == max(year)) %>%
  filter(total_jail_pop == max(total_jail_pop, na.rm = TRUE)) %>%
  pull(state)

highest_black_pop_state <- data %>%
  filter(year == max(year)) %>%
  filter(black_jail_pop == max(black_jail_pop, na.rm = TRUE)) %>%
  pull(state)

highest_black_pop_county <- data %>%
  filter(state == "CA") %>%
  filter(black_jail_pop == max(black_jail_pop, na.rm = TRUE)) %>%
  pull(county_name)
